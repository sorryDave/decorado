CSS es mas que estilo

La gran variedad de estilos, tecnicas y experimentos que ofrece CSS es todo un mundo en constante crecimiento

Esta pagina pretende animar a los participantes tanto a expresar su estilo y personalidad, como a aquel que unicamente quiera hacer un ejercicio de estilo o reorganizar la presentacion de los elementos del Html original

Una de las posibilidades mas interesantes que ofrece CSS es la de jugar con el contenido y reorganizarlo hasta el infinito, incluso dejando irreconocible el mensaje original o dando una vision nueva de este

Desde ocultar textos, imagenes hasta resaltarlas para una mayor legibilidad; Reorganizar elementos y componer un mensaje diferente o añadir animaciones e ilustraciones para redirigir la mirada hacia la parte importante o distraer al lector de esta misma

¿Para que sirve esta pagina?

Esta pagina es un laboratorio abierto a todo aquel que quiera experimentar con CSS.

Se ofrece un esqueleto básico de HTML5 para que los colaboradores maqueten y jueguen, todo esta permitido excepto manipular el propio fichero de Html.

En definitiva, es un campo de pruebas de todo lo que siempre quisiste hacer con hojas de estilo y nunca te atreviste, o un lugar donde mostrar tus habilidades y la potencia del diseño Web.

Tantos estilos como participantes

Todas las posibilidades, ideas que nunca probarias en un entorno de trabajo, los experimentos, la diversion, la seriedad y el profesionalismo se unen en esta pagina que solo pide una cosa: participacion

Participación, en efecto. Sin colaboradores, diferentes puntos de vista esta página no tiene sentido. Asi que animate y colabora una o las veces que quieras.
